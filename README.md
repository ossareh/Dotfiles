Dotfiles
========

Using the dotfiles from pythons pip repo. To get started on a new
machine run bootstrap.sh (curl it, run it, it sets up ~/Dotfiles as
part of it while ensuring basics are available)

### to add files:

`dotfiles --add ~/.vimrc`

### to sync

`dotfiles --sync -f`
